# embeddedos-org/.github

Org-wide configuration repo for the [EmbeddedOS](https://github.com/embeddedos-org)
GitHub organisation. Its only user-facing surface is **`profile/README.md`** — the
file GitHub renders at <https://github.com/embeddedos-org> as the org's "company
profile" page.

This repo deliberately contains **no source code**. For the deeper landing
index (full product catalogue, link tree, contribution table) see
[`embeddedos-org/embeddedos-org`](https://github.com/embeddedos-org/embeddedos-org).

## Files

| Path | Purpose |
|---|---|
| `profile/README.md` | Rendered as the org-profile page at <https://github.com/embeddedos-org>. Curated front-door. |
| `LICENSE` | MIT. |

## Editing the profile

The profile must stay short — it's a front door, not a manual. Anything that
needs more than ~5 minutes to read should live in
[`embeddedos-org/embeddedos-org/README.md`](https://github.com/embeddedos-org/embeddedos-org/blob/main/README.md)
instead and be linked from the profile.

## Contributing

Open a PR against `profile/README.md` for copy / link / badge changes.
Major restructures: please open an issue first.
